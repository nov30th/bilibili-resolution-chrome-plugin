# Privacy Policy / 隐私政策

**Last Updated / 最后更新: 2025-01-07**

---

## English Version

### Data Collection
This extension only collects and stores the following user preference settings:
- Whether to enable VIP quality selection (boolean value)

### Data Storage
All data is stored locally on your device using Chrome's local storage API (chrome.storage.sync).
We do NOT transmit any data to external servers.

### Data Usage
The collected data is only used for:
- Remembering your quality preference settings
- Automatically selecting the appropriate video quality for you

### Data Sharing
We do NOT share your data with any third parties.

### Data Security
All data is stored locally in your browser using Chrome's secure storage mechanism. No data is transmitted over the network.

### Data Deletion
Uninstalling this extension will automatically delete all stored data. You can also manually clear data by:
1. Going to `chrome://extensions/`
2. Finding this extension
3. Clicking "Remove" to uninstall

### Permissions Explanation
- **storage**: Used to save your preference settings (VIP quality toggle)
- **activeTab**: Used to interact with Bilibili video pages to change quality settings
- **host_permissions (bilibili.com)**: Required to access and modify video player settings on Bilibili

### Contact
If you have any questions, please contact: nov30th@gmail.com

## 中文版本

### 数据收集
本插件仅收集和存储以下用户偏好设置：
- 是否启用会员清晰度选择（布尔值）

### 数据存储
所有数据使用Chrome浏览器的本地存储API（chrome.storage.sync）存储在您的本地设备上。
我们不会将任何数据传输到外部服务器。

### 数据使用
收集的数据仅用于：
- 记住您的清晰度偏好设置
- 自动为您选择合适的视频清晰度

### 数据共享
我们不会与任何第三方共享您的数据。

### 数据安全
所有数据都使用Chrome的安全存储机制存储在您的浏览器本地。不会通过网络传输任何数据。

### 数据删除
卸载本插件将自动删除所有存储的数据。您也可以手动清除数据：
1. 访问 `chrome://extensions/`
2. 找到本插件
3. 点击"移除"卸载

### 权限说明
- **storage**：用于保存您的偏好设置（会员清晰度开关）
- **activeTab**：用于与B站视频页面交互以更改清晰度设置
- **host_permissions (bilibili.com)**：需要访问和修改B站视频播放器设置
